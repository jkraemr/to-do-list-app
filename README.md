# To-Do List App

Prototype of a simple to-do list app with the following features:

* Add a new item to list of items
* Cross out item from list of items
* Delete item from list of items
* Change order of items in list of items

## Built with
* HTML5
* CSS3
* JavaScript ES6

## Dependencies
* jQuery 3.5.1
* jQuery UI 1.12.1
